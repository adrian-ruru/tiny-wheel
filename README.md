# Tiny-Wheel

Es una aplicación web que trata principalmente sobre tirar de una ruleta con diferentes opciones.

Todo es personalizable, tanto las ruletas, sus nombres, las opciones de cada ruleta, los nombre de las opciones, sus porcentaje, colores, descripciones y más.

Además cuenta con multitud de funciones diferentes para explotar al máximo sus funcionalidades.

Actualmente seguimos en desarrollo de esta aplicación.
